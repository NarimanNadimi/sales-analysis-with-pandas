# Sales Data Analysis with Pandas & NumPy

این پروژه یک دیتاست فروش ساده را تحلیل می‌کند تا تمرین اولیه کار با Python، Pandas باشد.

## هدف پروژه
- تمرین خواندن فایل CSV با Pandas
- تحلیل داده‌ها و محاسبات عددی ساده با NumPy
- استفاده از GroupBy برای جمع‌بندی اطلاعات
- یادگیری نتیجه‌گیری و ارائه گزارش متنی

## دیتاست
- فایل: `sales.csv`
- ستون‌ها:
  - date: تاریخ فروش
  - product: نام محصول
  - city: شهر فروش
  - price: قیمت واحد
  - quantity: تعداد فروش

## آنچه انجام شد
- خواندن دیتاست و بررسی اطلاعات پایه
- محاسبه فروش کل هر ردیف
- جمع تعداد فروش کل
- جمع تعداد فروش بر اساس محصول و شهر
- پیدا کردن پرفروش‌ترین محصول و شهر
- نمایش بقیه شهرها و محصولات

## نحوه اجرا
1. نوت‌بوک `sales_analysis.ipynb` را باز کنید
2. سلول‌ها را به ترتیب اجرا کنید
